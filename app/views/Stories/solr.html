Success Story: Solr™ Lucene

Working as a Principal Research Scientist at Battelle Memorial Institute, Developer Dan Funk revamped a search engine for a large collection of research documents. Using the Apache Lucene™ library, Funk developed a powerful, full-text search engine. Dan additionally defined an OCR (optical character recognition) workflow process that could quickly produce very high quality, QA’d output while generating a custom XML format that allowed for fine-tuned search results. Ultimately search time was reduced from as much as 30 seconds to just a few milliseconds. And where the previous system had already been encountering scalability issues on large-scale servers, the new search system could comfortably run on a portable device.
